emacs init File: C-x C-f ~/.emacs.d/init.el RET
Undo: C-/ C-_ C-u
Cancel: C-g
Auto-complete: Tab
Go to beginning of a line: C-a end: C-e
Close file: C-x k

Fullscreen mode: toggle-frame-maximized
Tutorial Language: M-x help-with-tutorial-spec-language
load theme: M-x load-theme
